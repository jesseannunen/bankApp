# Basic Terminal Bank Application

A simple command-line banking application created as a learning project.

The application simulates basic banking functionality using a text-based user database.

---

## Features

- User login via username and PIN
- Balance check
- Simple account data stored in a text file
- Terminal-based user interface

---

## Requirements

- The application requires a file named `users.txt` to be present in the working directory.

### users.txt format

Each line represents a user account in the following format:

username PIN balance

Example:

john 1234 99
alice 4321 250

## Usage

Run the application from the terminal:

bash
./bank_app

## Purpose of the Project

This project was created to practice:

- basic file I/O

- user input handling

- simple application logic

- terminal-based program structure

It is intended as a small learning project rather than a production-ready application.
